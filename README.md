# SpeechFlow Web

SpeechFlow Web is a React + Vite frontend application that provides a clean interface for authentication, text-to-speech generation, and speech-to-text transcription.

## Features

- User Login Authentication
- Text-to-Speech Conversion
- Speech-to-Text Transcription
- Audio Playback Support
- Dashboard Interface
- API Integration with FastAPI Backend

## Tech Stack

- React.js
- Vite
- Axios
- JavaScript
- CSS

## Setup

```bash
git clone <repo-url>
cd speechflow_web
npm install
npm run dev
